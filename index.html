Política de Tratamiento de Datos

1. Responsable: Tecnología Para Todos procesa la información ingresada en el chat y la API Key con el único fin de proporcionar funcionalidad al sitio.

2. Finalidad: Los datos del chat se envían a los modelos de Google AI Studio para generar respuestas. La API Key se almacena localmente en su navegador (localStorage) y no se guarda en nuestros servidores.

3. Derechos: Usted puede eliminar su API Key en cualquier momento borrando los datos de navegación o desde el panel de configuración del sitio.

4. Herramientas de Terceros: Este sitio utiliza servicios de Google (Gemini API) y enlaces de afiliados (OptiSigns, Fliki). Le recomendamos revisar las políticas de privacidad de dichos proveedores.
